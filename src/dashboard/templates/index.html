      ⚙️ Konfigurasi USB

      Pisahkan dengan spasi untuk multiple ID

      📋 Petunjuk Deploy USB

      1. Colokkan USB yang sudah ditandai dengan file .cerberus_asist

      2. Sistem akan otomatis mendeteksi dan menjalankan setup

      3. Untuk membuat USB deploy: colok USB → klik "Siapkan USB Deploy"

      4. Atau jalankan manual: sudo bash scripts/usb/usb-trigger.sh /dev/sda1

      File yang diperlukan di root USB:
      .cerberus_asist (marker) — file kosong sebagai penanda
      cerberus_bundle.tar.gz (opsional) — bundle instalasi lengkap

      👤 Install SSH Key

      📖 Informasi SSH

      SSH keys terletak di: /opt/cerberus_asist/ssh-keys/

      Untuk remote access dari server lain:

      1. Generate keys di dashboard ini

      2. Copy public key ke remote server: ssh-copy-id user@remote-server

      3. Atau gunakan SSH tunnel: ssh -L 7860:localhost:7860 user@remote-server
